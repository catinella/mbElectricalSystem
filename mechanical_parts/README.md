## 2.0 Description:
This folder contains all CAD and specifications documents dequired to build the Motorbike Electrical System's containers. The
CAD documents are 2D drawings and they show you the flat parts you will have to weller. Anyway, read the README.md file in the
proper sub-folder for further details.

### 2.1 DR350 Rally
The motorbike I used as first model is a DR350 (1998) modified for off-road using. These changes include a couple of 48mm Showa
shock absorber and a motorcycle fairing made by [Twalcom](https://www.twalcom.com/en). So the front side box can be compatible
with many similar motorbikes. Instead, the box located in the read part is just for this motorbyke model. Infact, the box fills
the space originally used for the fuses-box, and the box's shape has been developed to be linked to the framework in a solid way.
In order to mount this part on your DR350, you need to make 5mm diameter hole in the old metal fuses-box support.
